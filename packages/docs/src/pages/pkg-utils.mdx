# @compiled/utils

General utilities used across packages.

## flatten

Flattens nested arrays into a single array.

```jsx
import { flatten } from '@compiled/utils';

flatten([1, [2, 3]]);
// > [1, 2, 3]
```

## unique

Returns an array with unique elements.

```jsx
import { unique } from '@compiled/utils';

unique([1, 1, 1, 2, 3]);
// > [1, 2, 3]
```

## hash

Hashes a string input to six characters.

```jsx
import { hash } from '@compiled/utils';

hash('hello-world');
// > "hqueq9"
```

## kebabCase

Transforms a string into `kebab-case`.

```jsx
import { kebabCase } from '@compiled/utils';

kebabCase('marginRight');
// > "margin-right"
```
