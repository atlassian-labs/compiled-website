# What is Compiled?

Compiled is a familiar and performant compile time [CSS-in-JS](https://reactjs.org/docs/faq-styling.html#what-is-css-in-js) library for [React](https://reactjs.org) with a two-step transformation that outputs [atomic CSS](/atomic-css).

## Create styles ahead of time

Write your styles in JavaScript or TypeScript and Compiled will **create all styles ahead of time** baking everything the component needs to run.

Resulting in **very performant components** that can be **used without configuration**,
import and go!

```jsx
import { EmphasisText } from '@component-library/text';

<EmphasisText>Hello world!</EmphasisText>;
```

## Extract styles in apps

Turn on extraction and all components styled in your app and sourced through NPM will have their **runtime stripped and styles extracted** to an atomic style sheet.

```diff
import { forwardRef } from 'react';
+import { ax, ix } from '@compiled/react/runtime';
-import { ax, ix, CC, CS } from '@compiled/react/runtime';
-const _3 = '._k48pni7l{font-weight:600}';
-const _2 = '._1p1dangw{text-transform:uppercase}';
-const _ = '._syazpjp7{color:var(--_ogjj5i)}';
export const EmphasisText = forwardRef(({ as: C = 'span', style, primary, ...props }, ref) => (
-  <CC>
-    <CS>{[_, _2, _3]}</CS>
    <C
      {...props}
      style={{ ...style, '--_ogjj5i': ix(primary ? '#00B8D9' : '#36B37E') }}
      ref={ref}
      className={ax(['_syazpjp7 _1p1dangw _k48pni7l', props.className])}
    />
-  </CC>
));
```

```css
._k48pni7l {
  font-weight: 600;
}
._1p1dangw {
  text-transform: uppercase;
}
._syazpjp7 {
  color: var(--_ogjj5i);
}
```

Resulting in **smaller JavaScript bundles** and even **better peformance**.
