# Installation

Setup is required.
Install the [React](https://reactjs.org) package with `npm` or `yarn`.

```bash
npm install @compiled/react
```

Compiled uses [Babel](https://babeljs.io) to transform your code,
so you'll need to use either [its CLI](https://babeljs.io/docs/en/babel-cli) or your bundler of choice.

## Webpack

Install the [Webpack](https://webpack.js.org/) loader.

```bash
npm install @compiled/webpack-loader --save-dev
```

Add the loader to your [Webpack config](https://webpack.js.org/).
If you're using [babel-loader](https://github.com/babel/babel-loader) ensure `@compiled/webpack-loader` is defined after.

```js
module.exports = {
  module: {
    rules: [
      {
        test: /\.js$/,
        exclude: /node_modules/,
        use: {
          loader: 'babel-loader',
        },
      },
      {
        test: /\.js$/,
        exclude: /node_modules/,
        use: {
          loader: '@compiled/webpack-loader',
        },
      },
    ],
  },
};
```

## Babel

Install the [Babel](https://babeljs.io) plugin.

```bash
npm install @compiled/babel-plugin --save-dev
```

Add the plugin to your [Babel config](https://babeljs.io/docs/en/configuration).

```json
{
  "plugins": ["@compiled/babel-plugin"]
}
```

> **Note** <br /> See the [plugin package documentation](/pkg-babel-plugin) for more information.

## Write your styles

Import the APIs you may already be familiar with and style away.

```jsx
import { styled } from '@compiled/react';

const ColoredText = styled.span`
  font-size: 11px;
  color: red;
`;
```
