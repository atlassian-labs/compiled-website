## Speed up your CSS-in-JS

Build with your bundler of choice or just [Babel](https://babeljs.io/docs/en/config-files),
resulting in **very performant components** that have their **styles built ahead of time**.

```json
{
  "plugins": ["@compiled/babel-plugin"]
}
```

```bash
$ babel src
Successfully compiled 1 file with Babel (9ms)
```

```jsx
import { CC, CS } from '@compiled/react/runtime';

const _2 = '._syaz1q9v{color: hotpink}';
const _ = '._1wybfyhu{font-size: 48px;}';

export const LargeHotPinkText = () => (
  <CC>
    <CS>{[_, _2]}</CS>
    <span className="_syaz1q9v _1wybfyhu">Hello world</span>
  </CC>
);
```

## Extract all styles to CSS

Turn on extraction and all components styled with Compiled in your app and sourced through [NPM](https://npmjs.com/) will have their styles extracted to a single [atomic style sheet](./docs/atomic-css).

1. Bundle your app

```bash
$ webpack
webpack 5.26.3 compiled successfully in 1912 ms
```

2. All styles and runtime are removed

```diff
-import { CC, CS } from '@compiled/react/runtime';
-
-const _2 = '._syaz1q9v{color: hotpink}';
-const _ = '._1wybfyhu{font-size: 48px;}';
-
export const LargeHotPinkText = () => (
-  <CC>
-   <CS>{[_, _2]}</CS>
    <span className="_syaz1q9v _1wybfyhu">Hello world</span>
-  </CC>
);
```

3. An atomic style sheet is generated

```css
._1wybfyhu {
  font-size: 48px;
}
._syaz1q9v {
  color: hotpink;
}
```
